Redshift vs Negative Blueshift

Redshift is represented by the letter z, and can be quantified by wavelength, frequency, or energy:

\[ 1 + z = \frac{wavelength_{obsv }}{wavelength_{emit}} \]
\[ 1 + z = \frac{frequency_{emit}}{frequency_{obsv }} \]


\[ 1 + z = \frac{Energy_{emit}}{Energy_{obsv }} \]

z↑    wavelength↑    frequency↓    Energy↓

z > 0 is redshift. z < 0 is blueshift.

Inverting the equations quantifies blueshift, represented by the letter b:

\[ 1 + b = \frac{1}{1 + z} \]
\[ 1 + b = \frac{wavelength_{emit}}{wavelength_{obsv }} \]
\[ 1 + b = \frac{frequency_{obsv }}{frequency_{emit}} \]


\[ 1 + b = \frac{Energy_{obsv }}{Energy_{emit}} \]

b↑     wavelength↓     frequency↑     Energy↑

b > 0 is blueshift. b < 0 is redshift.

The range of redshifts can be either infinite (as z) or finite (as -b):


z > 0
   
-1 < b < 0

This is useful.

The redshift-distance relationship (for z ≪ 1) is: \[ d = z \frac{c}{H_{0}} \] So the blueshift-distance relationship can be: \[ d = -b \frac{c}{H_{0}} \] Since: \[ 1 + b = \frac{1}{1 + z} \] \[ b = \frac{1}{1 + z} - 1 \] \[ b = \frac{1}{1 + z} - \frac{1 + z}{1 + z} \] \[ b = \frac{1 - (1 + z)}{1 + z} \] \[ b = \frac{-z}{1 + z} \] Therefore: \[ d = -b \frac{c}{H_{0}} \]
Becomes:
\[ d = \frac{z}{1 + z} \frac{c}{H_{0}} \]

Compare the two distance equations:

The redshift-distance equation climbs to infinity, while the blueshift-distance equation converges to c/H0.

Compare the blueshift-distance relationship to the concordance model of cosmology, λCDM.

The two lines are in the same ballpark.

Adjusting the parameters for λCDM removing:

  • Dark Energy (Ωλ=0)
  • The effects of gravity on redshifts (ΩM=0)

The two lines are a perfect match.
